# 原田式メソッド 目標達成アプリ "GoalCompass" 仕様書

## 1. 概要
本アプリは、「原田式メソッド」の核心である「長期目的・目標設定」と「オープンウィンドウ64（マンダラチャート）」をデジタル化し、目標達成を可視化・習慣化するためのWebアプリケーションです。
ユーザーは自分の目標を明確にし、それを具体的な行動計画（ルーティン）に落とし込み、日々の実行を管理することができます。

## 2. コンセプト & デザイン
- **コンセプト**: "To Achievement" - 達成を科学する
- **デザイン**: 
    - ユーザーが毎日開きたくなる「プレミアム」で「没入感」のあるデザイン
    - Apple風の洗練されたUI、Glassmorphism（すりガラス効果）、滑らかなマイクロインタラクション
    - 目標達成へのモチベーションを高める、視覚的に美しいデータ表現

## 3. 主要機能 (MVPスコープ)

### ① 目標設定 (Long-term Goal Setting)
「目的・目標の4観点」を用いて、目標に対する動機付けを強固にします。
- **メイン目標入力**: 達成したい目標と期日。
- **4観点入力**:
    1.  **私・有形**: 給料UP、欲しい物など
    2.  **私・無形**: 自信、スキル、健康など
    3.  **社会/他者・有形**: 売上貢献、チーム成果など
    4.  **社会/他者・無形**: 笑顔、信頼、良い雰囲気など

### ② オープンウィンドウ64 (Mandala Chart)
目標を具体的なアクションに分解します。
- **中心セル**: メイン目標（自動反映）
- **基礎思考（中核8マス）**: 達成に必要な8つの要素（心・技・体・生活など）
- **実践行動（外周64マス）**: 各要素を実現するための具体的な行動
- **UI/UX要件**:
    - 直感的な入力と、全体を見渡せるズームイン/アウト機能
    - 達成そのものを視覚化するプログレス表示

### ③ ルーティンチェック (Routine Check)
オープンウィンドウから導き出された行動を習慣化します。
- **ルーティンリスト**: 毎日行うべき行動のチェックリスト
- **実績ログ**: 継続日数や達成率の可視化
- **今日の一言**: 日誌機能（簡易版）

## 4. 技術スタック
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla/ES6+)
    - *理由*: 迅速なプロトタイピングと、将来的なフレームワーク移行の容易さ。複雑なビルド不要で即座に動作確認可能。
- **Data Persistence**: LocalStorage (初期フェーズ)
    - ブラウザを閉じてもデータが消えないようにローカル保存。

## 5. 画面構成案
1.  **Dashboard (Home)**:
    - 現在の目標と進捗率のサマリー
    - 本日のルーティンチェックウィジェット
2.  **Goal Editor**:
    - 4観点シートの入力・閲覧
3.  **Mandala View**:
    - 9x9のグリッドビュー。タップして詳細入力。
4.  **Routine / Journal**:
    - リスト形式のチェック画面
